## 📋 목표

업로드된 이미지와 동일한 디자인으로 전도인 카드 PDF 출력 양식 변경

## 🔍 현재 vs 목표 디자인 비교

### 제목
- **현재**: "회중용 전도인 기록 카드"
- **목표**: "회중용 전도인 기록"

### 상단 개인정보 섹션

#### 현재 레이아웃
```
이름: 박해숙          생년월일: 1954-02-25
성별: 여              침례일자: 1983-11-19
다른 양               장로, 봉사의 종, 정규 파이오니아
```

#### 목표 레이아웃 (이미지 기준)
```
성명:      박해숙                           ☐ 남    ☑ 여
생년월일:  1954-02-25                       ☑ 다른 양  ☐ 기름부음받은 자
침례일자:  1983-11-19
☐ 장로    ☐ 봉사의 종    ☑ 정규 파이오니아    ☐ 특별 파이오니아    ☐ 야외 선교인
```

**주요 차이점**:
1. 왼쪽에 레이블이 세로로 정렬 (성명, 생년월일, 침례일자)
2. 오른쪽에 체크박스들 배치 (남/여, 다른 양, 기름부음받은 자)
3. 직책 체크박스가 한 줄로 배치

### 테이블 구조

#### 현재 헤더
```
[봉사연도] | 참여 | 성서연구 | 보조 파이오니아 | 시간 | 비고
```

#### 목표 헤더 (이미지 기준)
```
봉사 연도 2026 | 봉사에 참여했음 | 성서 연구 | 보조 파이오니아 | 시간 (파이오니아 또는 선교인만 작성) | 비고
```

**주요 차이점**:
1. 첫 번째 열이 "봉사 연도 2026"으로 고정
2. "참여" → "봉사에 참여했음"
3. "시간" → "시간 (파이오니아 또는 선교인만 작성)"
4. 체크박스 스타일 (☑ / ☐)

## 🧪 검증 계획

### 수동 테스트

#### 테스트 1: 전도인 카드 PDF 생성
1. 로컬 개발 서버 실행: `npm run dev`
2. 브라우저에서 `http://localhost:5175/Ministry/` 접속
3. 관리자 계정으로 로그인
4. 대시보드에서 임의의 월 선택 → "상세" 클릭
5. 상세 페이지에서 메뉴(⋮) 클릭
6. "전도인카드 출력" 선택
7. 임의의 전도인 선택하여 PDF 생성
8. 생성된 PDF 확인:
   - ✅ 제목이 "회중용 전도인 기록"인지 확인
   - ✅ 개인정보 레이아웃이 이미지와 동일한지 확인
   - ✅ 체크박스가 ☑/☐ 스타일로 표시되는지 확인
   - ✅ 테이블 헤더가 이미지와 동일한지 확인
   - ✅ 테이블 데이터에 체크박스가 올바르게 표시되는지 확인

## 🎯 성공 기준

1. PDF 제목이 "회중용 전도인 기록"으로 표시
2. 개인정보 섹션이 이미지와 동일한 레이아웃
3. 체크박스가 ☑/☐ 스타일로 표시
4. 테이블 헤더가 이미지와 동일
5. 모든 데이터가 올바르게 표시
6. PDF가 정상적으로 생성 및 다운로드
