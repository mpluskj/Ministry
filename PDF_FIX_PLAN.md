# 전도인 카드 PDF 수정 계획

## 현재 문제점 (사용자 피드백)

1. **PDF 크기**: 가로 모드 → 세로 A4로 변경 필요
2. **폰트 크기**: 너무 작음
3. **레이아웃**: 정렬 및 간격이 맞지 않음
4. **체크박스**: 표시되지 않음 (폰트 문제)

## 수정 계획

### 1단계: PDF 방향 변경
- `jsPDF('l', 'mm', 'a4')` → `jsPDF('p', 'mm', 'a4')`
- 2개 위치 수정 필요

### 2단계: 여백 및 폰트 크기 조정
- 여백: 15mm → 20mm
- 제목 폰트: 18pt → 14pt
- 본문 폰트: 9pt → 10pt
- 테이블 폰트: 8pt → 9pt

### 3단계: 개인정보 섹션 레이아웃 수정
- 레이블 위치 조정
- 체크박스 위치 조정
- 줄 간격 조정 (7 → 8)

### 4단계: 테이블 시작 위치 조정
- startY 계산 수정
