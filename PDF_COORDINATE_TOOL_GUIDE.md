# PDF 좌표 조정 도구 사용 가이드

## 🎯 목적

S-21_KO.pdf 템플릿에 데이터를 정확한 위치에 배치하기 위한 좌표를 찾는 도구

## 🚀 실행 방법

### 방법 1: 브라우저에서 직접 접속

1. 브라우저 열기
2. 주소창에 입력:
   ```
   http://localhost:5175/Ministry/pdf-test
   ```
3. "좌표 조정 도구 열기" 버튼 클릭

### 방법 2: 대시보드에서 접속 (추후 추가 예정)

- 월별 상세 페이지 → 메뉴(⋮) → "PDF 좌표 조정"

## 📖 사용 방법

### 1. 도구 열기
- `/pdf-test` 페이지에서 버튼 클릭

### 2. 필드 선택
왼쪽 패널에서 조정할 필드 선택:
- 성명
- 생년월일
- 침례일자
- 남 체크박스
- 여 체크박스
- 다른 양 체크박스
- 기름부음받은 자 체크박스
- 장로 체크박스
- 봉사의 종 체크박스
- 정규 파이오니아 체크박스
- 특별 파이오니아 체크박스
- 야외 선교인 체크박스
- 테이블 시작 Y
- 참여 X
- 연구 X
- 보조 X
- 시간 X
- 비고 X

### 3. 좌표 조정

#### 슬라이더 사용
- X 슬라이더: 0 ~ 595 (A4 가로)
- Y 슬라이더: 0 ~ 842 (A4 세로)

#### 텍스트 입력
- 정확한 숫자 입력 가능

### 4. 실시간 미리보기
- 오른쪽 PDF 뷰어에서 즉시 확인
- "미리보기 새로고침" 버튼으로 수동 갱신 가능

### 5. 좌표 저장
- "좌표 내보내기 (JSON)" 버튼 클릭
- `pdf-coordinates.json` 파일 다운로드

## 📐 좌표 시스템

### PDF 좌표계
- **원점 (0, 0)**: 왼쪽 하단
- **X축**: 왼쪽 → 오른쪽 (0 ~ 595pt)
- **Y축**: 아래 → 위 (0 ~ 842pt)

### A4 용지 크기
- 가로: 210mm = 595pt
- 세로: 297mm = 842pt

### 좌표 찾기 팁
1. **대략적인 위치 추정**
   - 상단: Y = 750~800
   - 중단: Y = 400~600
   - 하단: Y = 100~300

2. **미세 조정**
   - 슬라이더로 대략 위치 찾기
   - 텍스트 입력으로 정확히 조정

3. **체크박스 위치**
   - 체크박스 중앙에 'X' 표시가 오도록 조정

## 💾 저장된 좌표 사용

### JSON 파일 형식
```json
{
  "name": { "x": 80, "y": 720, "label": "성명" },
  "birthDate": { "x": 80, "y": 705, "label": "생년월일" },
  ...
}
```

### 코드에 적용
1. JSON 파일 열기
2. 좌표 값 복사
3. `pdfTemplateOverlay.ts` 파일에 붙여넣기

## ⚠️ 주의사항

1. **폰트 문제**
   - 현재 Helvetica 폰트 사용 (한글 표시 안 됨)
   - 한글 폰트 임베딩 필요

2. **좌표 정확도**
   - 1pt 단위로 조정 가능
   - 너무 정밀하게 맞추려 하지 말 것

3. **테이블 행 간격**
   - `rowHeight` 값으로 조정 (기본 20pt)
